# Manual accessibility checks

- [ ] Skip link is the first keyboard focus and moves focus to `main`
- [ ] Desktop and mobile navigation work with keyboard only
- [ ] Language switching preserves the current page
- [ ] Focus remains visible at 200% and 400% zoom
- [ ] No horizontal page scroll at 320 CSS pixels
- [ ] Heading hierarchy and landmarks make sense in a screen-reader outline
- [ ] Selected-work and additional-work links have unambiguous names
- [ ] External-link behavior is announced to assistive technology
- [ ] Reduced-motion mode shows all content immediately
- [ ] Consent actions have equal prominence and can be changed later
- [ ] English and Czech pages expose the correct `lang` attribute
- [ ] Portrait, work, education, and logo alt text is appropriate in both languages
- [ ] CV text is selectable and its document structure is reviewed manually
