# Portfolio — cost & scaling

A static Next.js site on Vercel. The stack is in `about-project.md`; prices checked 2026-07-22.

## What it costs

- **Now:** $0/month + domain (Vercel Hobby; PostHog unconfigured; public-repo Actions free).
- **Commercial use:** $20/month + domain (Vercel Pro).

## When to scale

- Move to Vercel Pro when the site backs paid/freelance work — a licensing rule, not a traffic limit.
- Turn on PostHog only if you want analytics; it stays $0 well within 1M events/month.

## Keep costs down

Stay on Hobby while it's personal; set a Vercel spend alert if you move to Pro.
